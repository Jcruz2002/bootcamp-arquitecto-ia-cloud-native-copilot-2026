# Lab AZDO-05 - Boards y trazabilidad end-to-end

## Objetivo
Vincular código, PRs, pipelines y despliegues con Azure Boards.

## Prerrequisitos
- Labs AZDO-01 a AZDO-04 completados.

## Paso a paso
1. Crea Epics/Features/User Stories en Azure Boards.
2. Vincula commits y PRs a work items.
3. Configura policy para requerir work item link en PR.
4. Registra release notes con work items desplegados.
5. Construye dashboard de trazabilidad:
   - Work item -> PR -> Build -> Deploy

## Validación
- Cada cambio productivo tiene work item asociado.
- Dashboard muestra trazabilidad completa.

- Registra los resultados obtenidos en la carpeta labs/evidencias.

- Capturas de tablero y vínculos.
- Evidencia de release con work items cerrados.
