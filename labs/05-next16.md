# Lab 05 - Next 16 frontend conectado a API

## Objetivo
Construir UI funcional para listar y crear datos consumiendo backend.

## Prerrequisitos
- API disponible de Lab 03 o 04.

## Paso a paso
1. Configura variables de entorno de API.
2. Implementa pagina de listado.
3. Implementa formulario de alta.
4. Maneja estados de carga y error.
5. Agrega proteccion de rutas si aplica.

## Comandos sugeridos
```bash
cd templates/next16-app
npm install
npm run dev
```

## Validación
- Listado y alta funcionando desde UI.
- Errores de API mostrados de forma clara.

## Rúbrica
- 40% funcionalidad.
- 30% UX técnica (loading/error).
- 30% evidencia.
