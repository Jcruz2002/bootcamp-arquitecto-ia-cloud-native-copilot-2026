# Lab 06 - Angular 21 frontend alternativo

## Objetivo
Implementar un frontend Angular que consuma los mismos endpoints del backend.

## Prerrequisitos
- API disponible de Lab 03 o 04.

## Paso a paso
1. Genera servicios de acceso a API.
2. Crea componentes de listado y formulario.
3. Implementa validaciones de formulario.
4. Maneja errores HTTP de forma uniforme.
5. Documenta comparacion técnica vs Next.

## Comandos sugeridos
```bash
cd templates/angular21-app
npm install
npm start
```

## Validación
- CRUD basico disponible desde Angular.
- Validaciones y errores visibles.

## Rúbrica
- 40% funcionalidad.
- 30% arquitectura frontend.
- 30% evidencia.

## Entregables
- EVIDENCIAS.md y breve comparativa Angular vs Next.
