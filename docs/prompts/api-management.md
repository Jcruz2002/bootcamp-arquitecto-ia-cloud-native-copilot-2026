# Prompts para API Management (Azure APIM / Kong)

## Prompt principal
```text
Act as API Platform architect.
Publish this multi-service system through Azure APIM.
Requirements:
- API products and subscriptions
- JWT/OAuth validation policy
- rate limit and quota policies
- request/response transformation where needed
- versioning strategy
- developer portal readiness checklist

Return:
1. APIM design
2. policy examples
3. rollout plan by environment
4. validation tests
```

## Prompt comparativo
```text
Compare Azure APIM vs Kong for this project.
Return:
- decision matrix (security, operations, cost, ecosystem)
- recommendation by stage (MVP, growth, enterprise)
- migration path if switching later
```
